Tracking number configuration
SELECTION SITUATION FORMAT
The carrier do one parcel shipment only, shipment and parcel have the same tracking number. E.g. <check10><identity></check>
The carrier support multi parcel shipments, shipment and parcel have the same format but each parcel is identified with an additional number as well. E.g. <check11><sender.edi><identity></check>
The carrier have seperate formats for shipments and parcels, the parcel format is SSCC which we have support for. E.g. <check10><identity></check>
More advanced, e.g. different for different services. No format can be specified, it needs to be specified for a developer to write uniqe code.
